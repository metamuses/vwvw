Documentation

Project structure, stylistic choices, and technical implementation details.

1st step: content

Item research and selection: starting from the item used in the know org project we focused on three area of items related to Vivienne Westwood: punk 70s / XVIII sec France/ Manga,anime inspired. (n.b. third area expanded).
Project name brainstorming: wanted to work on the MMMM from the course project specs> ended up with Vivienne Westwood Virtual World

2nd step: structural choice

Project logo design inspo: VWVW per richiamare gioco parole progetto; forma logo ispirato a manifesto visto da marti; utilizzo delle lettere dal logo del brand Vivienne Westwood
In-real life museum organization: three rooms respected the themes behind the item selection (the idea is that the museum is still visitable without the web app- it has a items organization, and 3 possibile “narratives” based on the previously chosen themes)
Web-based narratives: the web app offers alternatives narratives for browsing the collection: historical> required (the original displayment of the item allows for a less intrigued visits of the collection in real life since the item are already organized based on an historical theme) made by Vivienne> a visits that allow to browse only the item in the museum vivienne made. This allows for a quicker and more “vertical” visits past and future heritage: a narratives that allows the user to browse between the item that inspired Vivienne’s work and the work that she has inspired.

3rd step: layout, wireframing

Starting from the website requirements from the course project specs we started sketching possible layout for the three main pages of the website that would be populated atm. COVER PAGE NARRATIVE PAGE ITEM PAGE

4th step: styles inspirations

The project specs required six different style sheets to be applied to the whole web project based on historic periods, from the general time-period requirment we moved on to more peculiar artistic inspiration. In particular the following are the chosen historical style inspirations:
1500-1800> the 700s in France (Corte di Re Sole)
XIX sec > invenzione del dagherottipo
1st half of the XX century> liberty e art noveau
2nd half of the XX century> punk
Late XX century- start of the XXI century > 90's italian cyberpunk
2035 > Dystopian anime future
